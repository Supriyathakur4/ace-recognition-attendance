 Face Recognition Attendance System with GUI

 Features

-  Face Detection using Haar Cascade
-  Face Recognition using ORB + BFMatcher
-  Attendance recorded in CSV
-  Voice confirmation using pyttsx3
-  Real-time GUI using Tkinter
-  Live list of students marked present

##  Requirements

- OpenCV
